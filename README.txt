How to capture on Windows:
  1. Open index.html in Chrome.
  2. Press F11 for fullscreen (optional, looks cleaner).
  3. Press Win + Shift + S, drag to select the area, it copies to clipboard.
  4. Open Paint, press Ctrl+V, then File -> Save As -> PNG into this folder with the name above.
